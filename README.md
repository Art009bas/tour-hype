# TourHype 🔥

**Прототип главной страницы — подборка горящих туров.**

Идея: главная страница туристического сайта должна быть не просто списком туров, 
а витриной с отобранными выгодными предложениями.

## Что реализовано

- 🏆 **Hero-секция** с ясным УТП: "Лучшие туры по реальным ценам"
- 🔥 **Бейджи на карточках**: "Горящий", "Выгода 49%", "Новинка"
- 💰 **Сравнение с конкурентами** прямо на карточке: "У Xottur: от 31 800 ₽ —28%"
- 🏷 **Зачёркнутая старая цена** + жирная новая
- 🎯 **Фильтры категорий**: Пляжные, Экскурсии, Горящие
- 📊 **Блок "Почему нам доверяют"** с цифрами
- 📞 **CTA-блок** "Не уверены, куда поехать?" с формой телефона
- 📱 Адаптивная вёрстка

## Локальный запуск

```bash
cd tour-hype
python3 -m http.server 8080
# → http://localhost:8080
```
